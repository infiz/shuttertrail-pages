Lightroom Classic plug-in

Match photos to GPX tracks, offset by offset.

Shutter Trail also provides a free Lightroom Classic plug-in that automatically geotags selected photos by matching their capture times to timestamped points in one or more GPX tracks. It is open source and hosted in infiz/shuttertrail-lightroom-gpx-sync on GitHub.

Download the latest release View the source on GitHub

Why use the plug-in?

Lightroom Classic’s built-in Map workflow can load a GPX tracklog, but it does not automatically read the UTC offset embedded in each selected photo. Instead, you adjust photo time with a single manual offset before applying the track locations.

That extra step is easy to get wrong, and one offset may not work for a selection containing photos from cameras set to different time zones. The result can be a location matched to the wrong point in the journey.

The plug-in reads each photo’s capture time, subsecond value, and embedded UTC offset with ExifTool. If an offset is missing, it asks what to use and suggests the most common offset found in the selection. It then finds the nearest timestamped GPX point, shows a summary for review, and updates the Lightroom catalog without modifying the original photo files.

Before installing

  • Use Adobe Lightroom Classic with Lightroom SDK 6.0 support or newer.
  • Use GPX tracks whose timestamps include Z or an explicit numeric UTC offset.
  • On macOS, install ExifTool first. In Terminal, run brew install exiftool. If needed, install Homebrew from brew.sh.
  • On Windows, the plug-in includes a portable ExifTool runtime.

Install the plug-in

  1. Open the latest release, download its source-code ZIP, and extract it.
  2. Move the complete shuttertrail-lightroom-gpx-sync.lrplugin folder to a permanent location. Keep all of its files together.
  3. In Lightroom Classic, choose File → Plug-In Manager.
  4. Select Add, choose the shuttertrail-lightroom-gpx-sync.lrplugin folder, and confirm the plug-in is enabled.

Geotag selected photos

  1. In Lightroom Classic’s Library module, select the photos to geotag. Video files are ignored.
  2. Choose Library → Plug-in Extras → Sync selected photos with GPX….
  3. Select one or more .gpx files, then choose Use GPX Files.
  4. If a photo has no embedded UTC offset, accept or change the suggested offset. Apply it to that photo, that camera, or all remaining photos without an offset—or skip those photos.
  5. Review the matching summary. Existing GPS data is preserved by default; enable replacement only when you intend to overwrite it.
  6. Confirm to write the matched latitude, longitude, and available altitude to the Lightroom catalog.

The closest GPX point must be within one hour of the photo’s timestamp. The plug-in does not interpolate between track points, and it rejects GPX timestamps that have no timezone information.
